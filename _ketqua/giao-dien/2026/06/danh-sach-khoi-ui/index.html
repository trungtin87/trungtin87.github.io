    Bảng Danh Sách Khối UI & Cấu Trúc HTML Tương Ứng

    Dùng để tham khảo khi xây thư viện component (component library) cho theme của bạn. Mỗi khối liệt kê các thành phần con (fields) thường gặp và thẻ HTML gợi ý.

    1. NHÓM ĐIỀU HƯỚNG (Navigation)

    Khối UI

    Thành phần con thường có

    Thẻ HTML gợi ý

    Header / Navbar

    logo, danh sách menu, nút CTA, icon menu mobile

    header, nav, ul > li > a, img, button

    Breadcrumb

    chuỗi liên kết phân cấp, ký hiệu ngăn cách

    nav, ol > li > a, span

    Sidebar

    tiêu đề nhóm, danh sách link, icon

    aside, nav, ul > li > a, svg

    Tabs

    danh sách nút tab, nội dung tương ứng

    div[role=tablist], button[role=tab], div[role=tabpanel]

    Pagination

    số trang, nút trước/sau

    nav, ul > li > a/button

    Footer

    logo, cột link, mạng xã hội, dòng bản quyền

    footer, nav, ul > li > a, p, svg

    2. NHÓM NỘI DUNG CHÍNH (Content Display)

    Khối UI

    Thành phần con thường có

    Thẻ HTML gợi ý

    Hero Section

    tiêu đề lớn, mô tả ngắn, nút CTA, ảnh/video nền

    section, h1, p, a/button, img hoặc video

    Article / Main Content

    tiêu đề, ngày đăng, tác giả, đoạn văn, ảnh minh hoạ, trích dẫn

    article, h1-h3, time, p, img, blockquote

    Card

    ảnh, nhãn/tag, tiêu đề, mô tả ngắn, link/nút

    div/article, img, span, h3, p, a/button

    Card Grid / List

    tập hợp nhiều Card

    div/ul (container) + nhiều Card

    Gallery / Portfolio

    ảnh, chú thích, overlay khi hover

    figure, img, figcaption

    Timeline

    mốc thời gian, tiêu đề sự kiện, mô tả

    ul/ol > li, time, h4, p

    Stats / Counter

    số liệu, nhãn mô tả

    div, strong/span (số), p (nhãn)

    Video Section

    tiêu đề, video nhúng, mô tả

    section, h2, iframe/video, p

    Table / Data Display

    tiêu đề cột, hàng dữ liệu

    table, thead > tr > th, tbody > tr > td

    3. NHÓM CHUYỂN ĐỔI / MARKETING (Conversion)

    Khối UI

    Thành phần con thường có

    Thẻ HTML gợi ý

    Features / Services

    icon, tiêu đề, mô tả ngắn (lặp lại nhiều lần)

    section, svg/img, h3, p

    CTA Section

    tiêu đề kêu gọi, mô tả ngắn, nút hành động

    section, h2, p, a/button

    Testimonials / Reviews

    avatar, tên, chức danh, nội dung đánh giá, số sao

    div, img, h4, span, blockquote/p, svg (sao)

    Pricing Table

    tên gói, giá, danh sách tính năng, nút chọn gói

    div, h3, span/p (giá), ul > li, a/button

    Partners / Logos

    dãy logo đối tác

    div/ul > li > img

    FAQ (Accordion)

    câu hỏi, câu trả lời ẩn/hiện, icon mũi tên

    details > summary, hoặc button[aria-expanded] + div

    Newsletter / Contact Form

    tiêu đề, input, nút submit, thông báo lỗi

    form, label, input/textarea, button, span.error

    Team Section

    ảnh, tên, chức danh, mạng xã hội

    div, img, h4, p, ul > li > a

    4. NHÓM TƯƠNG TÁC / TRẠNG THÁI (Interactive & Feedback)

    Khối UI

    Thành phần con thường có

    Thẻ HTML gợi ý

    Modal / Popup

    nền mờ (overlay), tiêu đề, nội dung, nút đóng

    div[role=dialog], div.overlay, h2, p, button

    Toast / Notification

    icon trạng thái, thông điệp, nút đóng

    div[role=status], svg, p, button

    Alert / Banner

    icon, thông điệp, nút hành động/đóng

    div[role=alert], svg, p, button

    Tooltip

    nội dung ngắn hiện khi hover

    span[role=tooltip]

    Dropdown / Select Menu

    nút mở, danh sách lựa chọn

    button, ul[role=listbox] > li[role=option]

    Chat Widget

    icon mở chat, khung chat, ô nhập, nút gửi

    div, button, div.messages, input, button

    Cookie Consent Banner

    nội dung thông báo, nút đồng ý/từ chối

    div[role=alert], p, button

    Search Bar

    input, icon kính lúp, nút submit

    form, input[type=search], svg, button

    Filter / Sort Controls

    nhãn, checkbox/select, nút áp dụng

    fieldset, label, input/select, button

    Loading / Skeleton

    khối placeholder mô phỏng nội dung

    div.skeleton (không có nội dung thật)

    5. NHÓM THƯƠNG MẠI ĐIỆN TỬ (E-commerce — nếu cần)

    Khối UI

    Thành phần con thường có

    Thẻ HTML gợi ý

    Product Card

    ảnh, tên sản phẩm, giá, giá gốc (gạch ngang), nút thêm giỏ

    article, img, h3, span.gia, del, button

    Cart Summary

    danh sách sản phẩm, số lượng, tổng tiền, nút thanh toán

    ul > li, input[type=number], p.tong-tien, button

    Product Detail

    gallery ảnh, tên, giá, mô tả, biến thể (size/màu), nút mua

    img/figure, h1, p, select/button[data-bien-the], button

    Rating / Review Input

    sao chọn, ô nhập nhận xét, nút gửi

    div.sao, textarea, button

    Gợi ý khi xây thư viện component

    Vì bạn viết tên hàm/biến bằng tiếng Việt không dấu hoặc có dấu theo kiểu tường minh, đây là ví dụ đặt tên nhất quán cho từng khối (dùng camelCase cho JS, kebab-case cho class CSS):

    tao_the_card(du_lieu_san_pham)       // JS: function tạo card sản phẩm
.khoi-hero                            // CSS class cho Hero Section
.khoi-cau-hoi-thuong-gap              // CSS class cho FAQ
moKhoiThongBao(thongDiep, loai)       // JS: mở toast/alert

    Mẹo tổ chức thư mục component:

    /components
  /khoi-dieu-huong   → header, nav, breadcrumb
  /khoi-noi-dung     → hero, card, gallery
  /khoi-chuyen-doi   → cta, testimonial, pricing
  /khoi-tuong-tac    → modal, toast, dropdown

    Mỗi khối nên tách thành 3 file riêng: .html (cấu trúc), .css (giao diện), .js (hành vi) — để dễ tái sử dụng và tùy biến độc lập khi đổi theme.
